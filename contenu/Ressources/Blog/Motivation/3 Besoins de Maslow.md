### Besoins de Maslow

- Titre : Identifier les besoins qui influencent la motivation
- Sous-titre : Un quiz pour reperer les besoins les plus actifs et mieux ajuster les priorites managériales.
- Statut : Beta
- Nombre de questions : 25
- Catégorie : Motivation

### À quoi sert ce quiz ?

- Texte : La pyramide de Maslow propose une lecture simple des besoins humains : securite, appartenance, reconnaissance, accomplissement. Dans une entreprise, ces besoins peuvent influencer la motivation, la disponibilite mentale, la confiance et l'envie de contribuer.

- Texte : Dans zzzbre, ce quiz aide a comprendre quel besoin semble le plus present a un moment donne, afin d'adapter les actions de leadership et les priorites d'accompagnement.

### Quand l'utiliser ?

- Texte : Ce quiz est pertinent pour reperer le besoin le plus actif chez une personne. Il aide a ajuster les priorites manageriales au niveau de motivation du moment.

### Ce que le quiz aide a comprendre

- liste :
  - Les besoins qui peuvent soutenir ou freiner la motivation.
  - Les signaux de securite, reconnaissance, appartenance ou accomplissement.
  - Les priorites d'accompagnement les plus utiles.
  - Les leviers humains a prendre en compte avant de demander plus de performance.

### Comment l'exploiter dans zzzbre

- Texte : Les resultats peuvent etre utilises pour preparer un echange, construire un plan d'action ou identifier des sujets collectifs a traiter. Ils doivent rester un support de dialogue, pas un diagnostic definitif.

### À retenir

- Texte : Avant de piloter la performance, il faut souvent comprendre les besoins qui conditionnent l'energie disponible, la confiance et l'engagement.
