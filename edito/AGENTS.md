# Instructions Agent

## Règle éditoriale

Toujours produire du contenu final, directement publiable par l'utilisateur final.

Ne pas écrire de note d'intention, de commentaire de conception, de justification, de méta-description ou d'instruction interne.

Le contenu doit apparaître comme il serait affiché dans l'interface ou sur la page :

- écrire les titres, paragraphes, libellés, CTA et liens finaux ;
- éviter les formulations comme `Objectif :`, `Positionnement :`, `Ce bloc sert à...`, `Texte :`, `Call to action :` ;
- ne pas expliquer pourquoi le contenu existe ;
- ne pas décrire le rôle du composant ;
- ne pas inventer de structure éditoriale non demandée.

### Exemple incorrect

- Objectif : Accompagner la lecture avec une sidebar orientée conversion.
- Texte : Créez un compte pour découvrir la plateforme.
- Call to action principal : Créer un compte.

### Exemple correct

#### Commencer avec zzzbre

Créez un compte pour découvrir les premiers modules beta et préparer un premier usage avec votre équipe.

[Créer un compte](https://zzzbre.com/login)
